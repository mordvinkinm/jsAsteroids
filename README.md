jsAsteroids       
===========       
         
Remake of "Asteroids" game, written on JavaScript.       
      
**Key map:**         
Arrow left	- turn counter-clockwise      
Arrow right - turn clockwise     
Arrow up	- throttle    
Space		- shoot   
    
F11			- full screen   
     
**Demo:**    
http://fixeg.github.io/jsAsteroids/
